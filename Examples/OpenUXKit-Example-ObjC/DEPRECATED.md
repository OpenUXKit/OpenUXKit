# DEPRECATED: OpenUXKit-Example-ObjC

This Objective-C example links against Apple's private `UXKit.framework` and
demonstrates a minimal navigation-controller scenario. It is kept in the
repository for historical reference only.

For a comprehensive, supported tour of the public **OpenUXKit** API surface
(views, bars, navigation, modal presentation, custom transitioning, the
collection view stack, appearance, layout guides, and the UXBase helpers),
build and run [`OpenUXKit-Example-Swift`](../OpenUXKit-Example-Swift) from the
top-level `OpenUXKit.xcworkspace`. Every demo there is wired up in pure Swift
against the SPM-exposed module.
